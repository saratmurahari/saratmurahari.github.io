# saratmurahari.github.io
React+Sass+Router v4.2.2 movie app

Installation


Install the dependencies and devDependencies and start the server.

$ npm install;


Run

$ npm run start;
Then navigate to http://localhost:8090

Build

$ npm run build;
//built files are exported to 'dist' folder
